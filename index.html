Estructura de datos: Arreglos

Arreglos Unidimensionales

Un arreglo(vector, array, matriz) es un conjunto de datos o una estructura de datos homogéneos que se encuentran ubicados en forma consecutiva en la memoria RAM (sirve para almacenar datos en forma temporal). Sus características son:

  • Finita: Todo arreglo tiene un límite, es decir se debe de determinar cuál será el número máximo de elementos que podrán formar parte del arreglo.
  • Homogénea : Todos los elementos de un arreglo son del mimo tipo.
  • Ordenada: Se puede determinar cuál será el primer elemento, el segundo el tercero y el último.

Los arreglos pueden ser de los siguientes tipos:

  -Arreglos unidimensionales o vectores ( Una dimensión)

  -Arreglos Bidimensionales o matrices (Dos dimensiones)

  -Arreglos Multidimensionales (Tres o mas dimensiones)

Pregunta

Seleccione el código necesario para declarar y dimensionar un arreglo unidimensional que permita
almacenar 10 calificaciones. 

Respuestas

int[] calificaciones = new int[];

int[] calificaciones = new int[10];

int calificaciones = int[10];

Retroalimentación

La forma correcta de declarar y dimensionar un arreglo  es:  int[] calificaciones = new int[10];

Hay que usar la palabra reservada new  e indicar el tamaño del arreglo.

Solución

  1. Incorrecto
  2. Correcto
